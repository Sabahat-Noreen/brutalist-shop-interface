# Amazon Homepage Clone

A pixel-perfect frontend clone of the Amazon homepage built entirely using vanilla **HTML5** and **CSS3**. This project replicates the structural layout, core navigation elements, and multi-card grid system of the e-commerce platform using custom assets.


## Key Features

* **Amazon-Style Header:** Replicates the signature dark navigation bar, including the search container, delivery location selector, language selector, accounts list dropdown link, and the shopping cart interface.
* **Sub-Navigation Menu:** Features the secondary category strip ("All", "Today's Deals", "Customer Service", etc.) for realistic visual mapping.
* **Dynamic Grid Layout:** Uses CSS Flexbox techniques to render responsive multi-card product clusters, displaying 12 distinct product/banner visual assets.
* **Clean UI Interactions:** Includes micro-interactions like crisp hover transformations on links, input focus styling, and product card shadows.
